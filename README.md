🏧 ATM Banking System

📌 About the Project

The ATM Banking System is a Python-based console application that simulates the basic functionality of a real ATM.

The program allows users to securely log in using a PIN, perform banking operations, and exit the system safely. It was built to practice Python fundamentals, loops, conditional statements, and menu-driven programming.

---

🚀 Features

✅ Secure PIN Verification

✅ Maximum 3 Login Attempts

✅ Account Lock After 3 Incorrect PINs

✅ Check Account Balance

✅ Deposit Money

✅ Withdraw Money

✅ Insufficient Balance Validation

✅ Menu-Driven Interface

✅ Exit Option

🎯 How It Works

1️⃣ User enters the ATM PIN.

2️⃣ Maximum 3 login attempts are allowed.

3️⃣ After successful login, the ATM menu is displayed.

4️⃣ The user can:

- Check Balance
- Deposit Money
- Withdraw Money
- Exit

5️⃣ The menu continues until the user selects Exit.

6️⃣ If the PIN is entered incorrectly three times, the account is locked.
